# Healthcare Dashboard

## Project Overview
Developed an interactive Healthcare Dashboard using Power BI to analyze hospital performance, patient admissions, revenue, insurance status, diagnosis trends, and readmission rates.

## Features
- Total Revenue Analysis
- Patient Admission Tracking
- Readmission Rate by Specialization
- Insurance Status Analysis
- Diagnosis-wise Patient Count
- Average Bill by Ward Type
- Hospital Ranking

- Power BI
- Excel
- Data Visualization
- DAX
